Bootstrap starter template

Hi Andrea, I suggest you pick whatever example page works best for your design and then customize it. You can leave the examples directory as is and just copy what you want to use into this _layouts directory.

As we discussed before, when you start to customize you can use more meaningful html 5 tags (ie header, footer, nav, section, aside.) than the bootstrap example files actually use :-). This helps make the divitis a little more manageable :-) But, really do whatever works for you! And use the online docs, they are really good.
